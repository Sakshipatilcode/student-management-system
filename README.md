# student-management-system
Beginner Full-Stack Student Management System built using Django, HTML, CSS, and JavaScript.
## Features

- User Registration
- User Login / Logout
- Add Students
- Edit Students
- Delete Students
- Search Students

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Django
- Django REST Framework

Database:
- SQLite
